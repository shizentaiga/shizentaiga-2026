## 🛡️ 検証プロトコル
すべての新機能は以下のステップを経て本番公開されます。
1. **Local**: `npm run dev` でのロジック確認。
2. **Preview**: `npm run deploy` 後の `*.workers.dev` での実機・レスポンシブ確認。
3. **Production**: プレビュー環境でのサインオフ後、本番反映。
